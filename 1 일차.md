#### 기본 연산자

```ruby
puts 2+3 # => 5 더하기
puts 3-1 # => 2 뺄셈
puts 3*2 # => 6 곱셈
puts 3/3 # => 1 몫
puts 3**2 # => 9 제곱
puts 3%2 # => 1 나머지

```



####  비교 `puts` vs `print`

```ruby
print "Hello!" # 개행 없음 java print
puts "Hello!"  # 개행 있음 java println
# => Hello!Hello!

puts "Hello!"
print "Hello!"
# => Hello!
#    Hello!
```
