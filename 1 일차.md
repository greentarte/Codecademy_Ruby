#### 기본 연산자

```ruby

```



####  비교 `puts` vs `print`

```ruby
print "Hello!" # 개행 없음 java print
puts "Hello!"  # 개행 있음 java println
# => Hello!Hello!

puts "Hello!"
print "Hello!"
# => Hello!
#    Hello!
```
